### 一、相关技术
- 前端：`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 首页、3. 文章、4. 心理评测、5. 留言、6. 公告

管理员功能：1. 登录、2. 文章查询、3. 添加文章、4. 留言列表、5. 用户列表、6. 测评题目查询、7. 测评管理、8. 公告查询、9. 公告添加
